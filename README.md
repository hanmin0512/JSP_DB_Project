# my_project
### DB 프로젝트 웹 어플리케이션
## 사용한 언어 java
- java의 JSP를 사용하여 웹 어플리케이션을 구현하였습니다.

## 사용한 DB ORACLE DB
- oracle 회사의 ORRACLE DB를 사용하였습니다.
- 학교에서 사용하는 UNIX DB서버 이용

## aws의 ubuntu 서버에 WAS 운영
- apache-tomcat8.5를 사용하여 WAS 운영하였습니다.
